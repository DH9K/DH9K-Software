> [!NOTE]

> 1. On this repo click Code > Download ZIP
> 2. Extract Downloaded File
> 3. Open Newly Extracted Folder
> 4. Finally Open "index.html"

> Hosting:<br>

> 2. Open the new fork
> 3. Go to Settings > Pages
> 4. Finally under branch select "main"
